⚡ QUICK START - 5 MINUTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Update API URLs (1 minute)
   $ sed -i '' 's|http://localhost:8000/api|/api|g' app.js pages/*.html

Step 2: Push to GitHub (1 minute)
   $ git init
   $ git add .
   $ git commit -m "Deploy to Vercel"
   $ git remote add origin https://github.com/USERNAME/investment-site.git
   $ git push -u origin main

Step 3: Deploy to Vercel (1 minute)
   $ npm install -g vercel
   $ vercel --prod

Step 4: Set Environment Variables (1 minute)
   Go to Vercel Dashboard → Settings → Environment Variables
   Add:
   • MONGODB_URI = (your connection string)
   • JWT_SECRET = (generate with node -e "...")
   • FRONTEND_URL = https://your-project.vercel.app

Step 5: Done! 🎉 (1 minute)
   Visit: https://your-project-name.vercel.app

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: Do I need to change my MongoDB?
A: No! Your MongoDB Atlas stays the same. Vercel just connects to it.

Q: Will my data transfer?
A: No need - everything stays in MongoDB Atlas. Vercel is just your web server.

Q: Can I use a custom domain?
A: Yes! Add it later in Vercel Dashboard → Settings → Domains

Q: What if something breaks?
A: Use `vercel logs` to check errors, then `vercel rollback` if needed

Q: Do I need to change my code much?
A: Only update API URLs from localhost:8000 to /api (one sed command!)

Q: How much will it cost?
A: Vercel has a free tier! Upgrade only if you need more resources.
